## Instruções de Produção

1. **Respeitar Ordem Rigorosamente**
   - Blocos 1-7 formam sequência pedagógica coerente
   - Bloco 8 inicia novo módulo (irmão de 2, filho de 1)
   - Blocos 9-14 formam submódulo de Cirurgia Vascular

2. **Usar Diretrizes ANIMA**
   - Ler `anima_filosofia.md` para tom e pedagogia
   - Ler `anima_diretrizes_producao.md` para estrutura
   - Ler `anima_bloco_canonico.md` para template de referência

3. **Lentes Pedagógicas**
   - `visao_geral`: Overview genérico, não-técnico, motivacional
   - `fisiopatologia`: Mecanismos de doença, causa-efeito
   - `apresentacao_clinica`: Sinais, sintomas, história, exame
   - `diagnostico`: Como confirmar/diagnosticar
   - `conduta`: Como tratar/intervir
   - `correlacao_basica`: Ligar anatomia/fisiologia básica

4. **Contexto e Escopo**
   - Cada bloco tem escopo delimitado no campo "Contexto"
   - NÃO expandir além do escopo definido
   - Se houver dúvida, consultar bloco pai para contexto maior

5. **Profundidade**
   - Profundidade 1-2: visão geral, grande cobertura, menos detalhe
   - Profundidade 3: temas específicos, detalhado
   - Profundidade 4+: folhas, muito detalhado, especializado
